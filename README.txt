WHITE LOTUS DETAILS - PAGINA DE AFILIADOS

COMO ABRIRLA
1. Abre la carpeta.
2. Haz doble clic en index.html.

COMO AGREGAR PRODUCTOS
1. Abre products.js con Bloc de notas o cualquier editor de texto.
2. Busca la categoria: herramientas o materiales.
3. Copia el ejemplo y cambia nombre, enlace y descripcion.
4. Guarda el archivo.

EJEMPLO:
{ nombre: "Nombre del producto", enlace: "https://amzn.to/...", descripcion: "Descripcion breve." }

IMPORTANTE
- Separa cada producto con una coma.
- No borres los corchetes [ ].
- El aviso de afiliacion ya esta incluido al final de la pagina.

PUBLICACION GRATUITA
Puedes publicar la carpeta gratis usando GitHub Pages, Netlify o Cloudflare Pages.
